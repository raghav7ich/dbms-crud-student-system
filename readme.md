## Description
This is a CRUD (Create, Read, Update, Delete) application developed using Flask and MySQL.

## Technologies Used
- Python (Flask)
- MySQL
- HTML, CSS

## Features
- Add student
- View students
- Update student
- Delete student

## How to Run
1. Install dependencies:
   pip install flask mysql-connector-python

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
